Ce module utilise des données du module "gestion des inventaires", il peut mettre à jour automatiquement la liste des machines:
- dans Remote Desktop Manager
- dans l'annuaire de RDP (Sur IE: cliquez ici, sur Chrome ou FireFox: buton droite et copies l'adresse du lien et coller sur Windows explorer, trouvez votre seveur dans Windows explorer et les executer avec powershell)

Il utilise 3 colonnes de inventaire ("full name", "canonical name" et "in service") pour créer la liste.
Nous pouvons ajouter les equipements comme les switches par example directement dans inventaire.
"canonical name" definit les emplacements, "full name" pour la connexion et "in service" pour ajouter ou enlever.
