Pour l'équipe "SSA - Support administratif"


  Étape 1

        Sous le menu "Rétrofacturation du compte bilan", sélectionnez la fonction "Importation".


        Cette fonction construira une liste à partir des CI Octopus ayant l'attribut "RF - Centre de coûts" documenté et l'attribut "RF - Complétée" vide.

        Cette liste sera visible automatiquement dans le vue "Validation".


  Étape 2

        Sous le menu "Consulter la vue "Validation" afin de faire une assurance qualité des équipements identifiés pour la rétrofacturation.


        Une entrée est adéquate si elle possède un code budgétaire conforme et un coût valide (il ne devrait pas y avoir de coût à 0$).

        Pour obtenir la liste des items pour lesquels le code budgétaire est non conforme, cliquer sur le lien "Cliquez ici pour filtrer seulement le code budgétaire non conforme ou le coût vide (0.00 $)".

        Différentes recherches sont possibles en cliquant dans le champ SEARCH et en spécifiant un critère de recherche.


  Étape 3

        Sous le menu "Rétrofacturation du compte bilan", sélectionnez la fonction "Exportation".


        La vue "Entrepôt de fichiers" sera affichée automatiquement.

        Cliquer sur le lien "fichier pour la comptabilité en csv" de la dernière exportation (tri possible sur la colonne date).

        Le fichier sera téléchargé localement sur votre poste de travail.

        Envoyer ce fichier par courriel à la direction des ressources financières via l'adresse courriel « 06 CEMTL Comptabilité/Reg06/SSSS ».
