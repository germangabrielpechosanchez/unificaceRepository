Selectionnez le menu GESTION DE MAPPAGE D'IMPRIMANTES


  Assigner / Désassigner des imprimants aux emplacements ( aussi les imprimants par défaut)

        Sous le menu "Assigner / Désassigner des imprimants", sélectionnez le button: +ADD RECORD / EDIT


  Assigner / Désassigner des clients légers aux emplacements

        Sous le menu "Assigner / Désassigner des clients légers", sélectionnez le button: +ADD RECORD / EDIT


  Ajouter / Activer / Désactiver des imprimantes

        Sous le menu "Ajouter / Activer /Désactiver des imprimantes", sélectionnez le button: +ADD RECORD / EDIT


  Ajouter / Activer / Désactiver des Clients légers

        Sous le menu "Ajouter / Activer / Désactiver des Clients légers", sélectionnez le button: +ADD RECORD / EDIT


  Ajouter / Activer / Désactiver des emplacements

        Sous le menu "Emplacements", sélectionnez le button: +ADD RECORD / EDIT


  Vu ensemble des client légers - imprimantes

        Sous le menu "Vue client légers - imprimantes", validation visuelle avant l'application


  Appliquer les changement au script

        Sous le menu "Exporter au script", Cette étape est requise pour enregistrer les modifications
