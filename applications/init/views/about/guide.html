Unifier pour être éfficace

C'est quoi?

“You don't have to cook fancy or complicated masterpieces - just good food from fresh ingredients.” ― Julia Child

Basé sur les ingrédients suivants: web2py, python, PowerShell et SQL Server

  1. web2py  Un Framework Open Source complet sans dépendance, simple, sécure et moderne
  2. python Un langage lisible, populaire, librairie riche et multiplateforme
  3. PowerShell Conçu sur .NET Framework, bien intégré dans l'écosystème de Microsoft, open source et multiplateforme
  4. SQL Server Une base de données bien connue et nous avons déjà en place

Une version évolutive de timeWalker 

  1. Conception sur meusure, interface simple, stable depuis sa sortie en 2008
  2. Collaboration possible avec une autre application, comme Virtuo MediAccès
  3. Ses utilisateurs sont très satisfaits

Pourrait être utile dans:

  • Depuis: manipulation des paquets réseau
  • Jusqu'à : accès à des bases de données en nuage
